# projX-la-7zip

## 项目名称

7zip的LoongArch优化

## 支持单位

龙芯中科技术股份有限公司、中国科学院计算技术研究所

## 项目描述

为开源的7zip压缩软件进行LoongArch架构的针对性优化，实现类似X86/Arm等架构的汇编优化，利用SIMD指令等架构提供的功能进行细致调优，大幅度提升龙芯平台的压缩效率。
